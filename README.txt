How to run the game:
--------------------

Requires
 - Windows 7 or later, or
 - Mac OS X 10.6 or later

 - A DirectX 9 compatible graphics card

 - Python 2.7 or later
 - Bacon 0.3.1, install from https://pypi.python.org/pypi/bacon/0.3.1

Run:

	python.exe run_game.py
